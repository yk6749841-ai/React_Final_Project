# פרויקט React

## סקירה כללית
הפרויקט הוא אפליקציית ווב מבוססת React לניהול כרטיסים, משתמשים, עדיפויות ותגובות. האפליקציה מספקת ממשק ניהול מקיף למנהלים, סוכנים ולקוחות.

## תכונות
- **אימות משתמשים**: מערכת כניסה והרשמה מאובטחת.
- **לוח מחוונים**: סקירה כללית של כרטיסים, משתמשים ועדיפויות.
- **ניהול כרטיסים**: יצירה, עדכון, הקצאה ומעקב אחר כרטיסים.
- **ניהול משתמשים**: הוספה, עדכון וניהול תפקידי משתמשים והרשאות.
- **תגובות**: הוספה וצפייה בתגובות על כרטיסים.
- **עדיפויות**: ניהול עדיפויות לכרטיסים.
- **שומרי דרך**: הגנה על נתיבים עם שומרי כניסה והרשאות.

## מבנה הפרויקט
הפרויקט מאורגן בתיקיות הבאות:

- **src**: מכיל את קוד המקור.
  - **components**: רכיבי React לשימוש חוזר כמו `dashboard`, `login`, `register` ועוד.
  - **features**: Redux slices לניהול מצב (לדוגמה, `authSlice`, `commentsSlice`).
  - **models**: מודלים של TypeScript למבני נתונים (לדוגמה, `user`, `tickets`).
  - **services**: שירותי API לתקשורת עם השרת (לדוגמה, `authenticationApi`, `ticketApi`).
  - **api**: יירוט טוקן לאבטחת קריאות API.
  - **assets**: נכסים סטטיים כמו תמונות וסגנונות.

- **public**: מכיל קבצים סטטיים כמו `index.html`.
- **project**: קבצי קונפיגורציה כמו `eslint.config.js`, `tsconfig.json`, ו-`vite.config.ts`.

## טכנולוגיות בשימוש
- **React**: פריימוורק צד לקוח.
- **TypeScript**: להבטחת סוגי נתונים.
- **Redux Toolkit**: לניהול מצב.
- **Vite**: כלי בנייה.
- **ESLint**: בדיקת קוד.

## התקנה
1. שיבטו את הריפוזיטורי:
   ```bash
   git clone https://github.com/yk6749841-ai/React_Final_Project.git
   ```
2. עברו לתיקיית הפרויקט:
   ```bash
   cd project
   ```
3. התקינו את התלויות:
   ```bash
   npm install
   ```
4. הפעילו את שרת הפיתוח:
   ```bash
   npm run dev
   ```

## שימוש
- פתחו את האפליקציה בדפדפן בכתובת `http://localhost:3000`.
- התחברו או הירשמו כדי לגשת ללוח המחוונים.
- השתמשו בתפריט הניווט לניהול כרטיסים, משתמשים, עדיפויות ותגובות.
